# Cloudflare Skills

根据 [Cloudflare 开发者文档](https://developers.cloudflare.com/) 创建的 skills 集合。

## 包含的 Skills

| Skill | 说明 |
|-------|------|
| `cloudflare-agents` | Cloudflare Agents SDK 完全指南 — Agent 类、Think 框架、状态管理、调度、工具、子代理等 |
| `workers-ai` | Cloudflare Workers AI 完全指南 — 模型目录、Workers 绑定、Vercel AI SDK 集成、REST API、定价 |

## 使用方法

将本目录链接到你的 agent 配置即可。
