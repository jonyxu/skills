# 中国传统色配色工具（chinese-colors）

基于中国传统色彩体系，提供专业配色方案生成与色彩分析。

## 使用场景

- UI/UX 设计配色
- 品牌视觉系统搭建
- 插画/国风设计配色
- PPT/演示文稿配色
- 室内/空间色彩设计
- 服装/穿搭色彩搭配

---

## 配色原理

### 60-30-10 法则

| 占比 | 角色 | 说明 |
|------|------|------|
| 60% | 主色 | 页面背景、大面积区域 |
| 30% | 辅色 | 次要区域、卡片、侧边栏 |
| 10% | 强调色 | 按钮、链接、高亮 |

### 对比配色类型

| 类型 | 说明 | 情感效果 |
|------|------|---------|
| 互补色 | 色轮上180度对角 | 强烈对比、视觉冲击 |
| 类比色 | 色轮上相邻30度 | 和谐统一、舒适自然 |
| 三色组 | 色轮上120度等分 | 丰富平衡、活泼动感 |
| 分裂互补 | 主色+互补色两侧 | 对比柔和、优雅精致 |

## 配色工作流

### 步骤 1：确定主题情感

| 情感目标 | 推荐色系 | 关键词 |
|---------|---------|--------|
| 热情/活力 | 红色系、橙色系 | 朱砂、落霞、珊瑚 |
| 专业/信任 | 蓝色系 | 石青、靛蓝、景泰蓝 |
| 自然/健康 | 绿色系 | 松柏、青葱、竹青 |
| 高贵/奢华 | 紫色系、金色 | 魏紫、赤金、绛紫 |
| 温暖/亲和 | 黄色系、暖色 | 栀子、姜黄、琥珀 |
| 清新/宁静 | 蓝绿色系 | 月白、天青、霜绿 |
| 古典/传统 | 红绿金组合 | 朱砂、青葱、赤金 |

### 步骤 2：选择配色方案类型

| 场景 | 推荐方案 | 说明 |
|------|---------|------|
| 登录页/首页 | 互补色对比 | 石青x栀子、朱柿x蓝采和 |
| 产品详情页 | 类比色和谐 | 月白x天青、云雀x霜绿 |
| 国风/节庆 | 红绿经典 | 海棠x青葱、朱砂x竹青 |
| 品牌VI | 五色方案 | 故宫红墙、青花瓷韵 |
| PPT/演示 | 三色组合 | 景泰蓝+月白+赤金 |

### 步骤 3：应用配色比例

**UI 设计推荐比例：**
- 背景色（主色 60%）：素色 #F5F5F5 或月白 #D6ECF0
- 卡片/容器（辅色 30%）：白色或浅色系
- 按钮/链接（强调色 10%）：朱砂 #FF461F 或石青 #1E80A0

**插画/国风设计推荐比例：**
- 背景（50%）：素色或淡雅色
- 主体（30%）：主题色
- 点缀（15%）：对比色
- 细节（5%）：强调色

### 步骤 4：验证配色效果

**检查清单：**
- [ ] 对比度是否满足 WCAG AA 标准（4.5:1）
- [ ] 主色、辅色、强调色比例是否合理
- [ ] 色盲用户是否能区分关键信息
- [ ] 在不同设备上显示是否一致
- [ ] 与品牌调性是否匹配
